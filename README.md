# OpenClaw 协作知识库

这是 Kevin 与图灵（AI 助手）的协作笔记仓库。

## 目录结构

| 目录 | 用途 |
|------|------|
| `inbox/` | 快速记录的想法、碎片，待整理 |
| `knowledge/` | 整理后的知识卡片 |
| `projects/` | 项目相关笔记 |
| `daily/` | 日记（可选） |
| `templates/` | 模板文件 |

## 工作流

1. Kevin 在 `inbox/` 记录想法
2. 图灵定时整理 → 移动到 `knowledge/` 或 `projects/`
3. 图灵也可以主动生成内容
4. 双方通过 Git 同步
